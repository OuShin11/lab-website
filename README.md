# Laboratory Website

## Overview
本プロジェクトは、研究室の情報を分かりやすく発信することを目的としたWebサイトである。  
研究内容、メンバー情報、イベントなどを整理し、閲覧しやすい構成で設計・実装を行った。

## Features
- 研究室の情報を整理した複数ページ構成
- PC・スマートフォンの両方に対応したレスポンシブデザイン
- シンプルで可読性を重視したUI設計
- 各ページ間のナビゲーションを整備し、直感的に操作可能

## Pages
- index.html：トップページ
- research.html：研究内容の紹介
- theses.html：卒業研究テーマ
- event.html：イベント情報
- supplement.html：補足情報
- contact.html：連絡先

## Background
研究室の情報を整理して外部に発信するために、個人でWebサイトの設計・実装を行った。  
単に情報を掲載するだけでなく、閲覧者が必要な情報にすぐアクセスできるよう、ページ構成や導線設計を工夫した。
